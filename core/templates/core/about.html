About Cloud Devopsec Project

This project is being done by Ali Hamza 23257164 for the course of cloud devopsec and involves using Amazon Web Services and other cloud services to create a project that is secure and reliable. The project aims to be used for real-world applications and includes a pipeline using AWS CodePipeline and other services for secure and reliable deployment. The application provides basic CRUD operations functionality.

Deployment

The application is deployed using Elastic Beanstalk, providing a scalable and reliable environment for hosting. Continuous integration and code quality checks are performed using SonarCloud, ensuring high-quality code and security standards.

Features

  • Accepts user input and validates it
  • Provides CRUD functionalities
  • Utilizes Amazon Web Services for deployment and other cloud services for reliability
  • Secure deployment using AWS CodePipeline and Elastic Beanstalk
  • Code quality checks with SonarCloud
